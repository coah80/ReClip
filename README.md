# ReClip

youtube removed clips so i added it back

ReClip is a Chrome and Firefox extension that restores the Clip button on YouTube watch pages while YouTube's native clip creation flow still exists.

## Install on Chrome

1. Download or clone this repo.
2. Open `chrome://extensions`.
3. Turn on Developer mode.
4. Click Load unpacked.
5. Select the `ReClip` folder.
6. Open a YouTube video while signed in.
7. Click the scissors Clip button next to YouTube's action buttons.

## Install on Firefox

1. Download and unzip the release.
2. Open `about:debugging#/runtime/this-firefox`.
3. Click Load Temporary Add-on.
4. Select `manifest.json` inside the unzipped `ReClip` folder.
5. Open a YouTube video while signed in.
6. Click the scissors Clip button next to YouTube's action buttons.

## Controls

Click the ReClip extension icon to turn the restored Clip button on or off.

## Notes

ReClip does not run a server, collect analytics, or store anything except the local on/off setting for the Clip button.
