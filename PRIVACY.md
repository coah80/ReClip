# ReClip Privacy Policy

ReClip does not collect, sell, share, or transmit personal data.

The extension runs only on YouTube pages so it can restore YouTube's native Clip button and clip creation panel. ReClip stores one local browser setting: whether the injected Clip button is enabled. That setting stays in the browser through Chrome extension storage.

ReClip does not use analytics, tracking, remote logging, advertising identifiers, or external servers.

ReClip is not affiliated with YouTube or Google.
